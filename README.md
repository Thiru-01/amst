# AmSt 
<img src="screenshots/icon.png" width="70"/>

*A messaging application which uses Firebase for backend and Getx for statemanagement having minimalstic user interface*

## Features
1. Chatting with Peer
2. Able to send image
3. Notifications
4. Last seen
